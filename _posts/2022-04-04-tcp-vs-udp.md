- HTTP is an application-layer protocol that depends on lower-level protocols like TCP/UDP
- **TCP** is connection-oriented via a 3-way handshake; has congestion control; uses checksum to guarantee corruption-free data transfer
- **UDP** is faster but can lose data and has no congestion control — used for streaming, gaming, etc.
