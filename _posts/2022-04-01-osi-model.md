The OSI model has 7 layers. Mnemonic: **Please Do Not Throw Sausage Pizza Away** (Physical → Application).

| Layer | Name | Role |
|-------|------|------|
| 7 | Application | What users interact with (HTTP, FTP) |
| 6 | Presentation | Formats data for the application layer |
| 5 | Session | Establishes sessions between ports |
| 4 | Transport | TCP/UDP; message segmentation and acknowledgement (3-way handshake) |
| 3 | Network | Packets containing IP addresses |
| 2 | Data Link | Frames (like envelopes) containing MAC addresses; transfers data error-free; NIC → switch → NIC |
| 1 | Physical | Information converted to bits and transmitted via cable |
