- **.NET Core** is not the same as **.NET** (which refers to .NET Framework, now just called .NET)
- **ASP.NET 4.x** is now called **ASP.NET** — the naming has been cleaned up with .NET 5+

## Testing Goals

- **Frontend:** test using DevTools and Redux DevTools for networking, elements, and state management
- **Backend:** learn how to performance-test and load-test individual API endpoints
