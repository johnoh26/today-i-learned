- Think of a Turing machine as a finite state machine with an infinitely long tape
- It can read from and write to the tape
- It can operate at any arbitrary position on the tape
- Behavior depends on the current state and what's on the tape — based on that, it writes to the tape (or leaves it unchanged) and moves left or right
